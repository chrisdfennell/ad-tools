About Token Size

The Kerberos ticket contains Security Identifiers (SIDs) for every group the user belongs to, including nested groups.

Thresholds:

  • OK Under 12,000 bytes
  • Warning 12,000 - 48,000 bytes. May cause HTTP authentication failures.
  • Critical Over 48,000 bytes. Exceeds default MaxTokenSize.

Estimation formula:

1200 + (groups x 40) + (domain_local x 40)

Remediation: Remove unnecessary group memberships, consolidate groups, or increase MaxTokenSize via GPO.
